# AoC 2022

Attempting a different language every day ... again.

Each day can be run using
```docker build -t <day> <day>/ && docker container run -it --rm <day>```

E.g. ```docker build -t one one/ && docker container run -it --rm one```

* 2 - Rust
* 3 - Racket
* 4 - PHP
* 5 - Erlang
* 6 - C++
* 7 - Julia
